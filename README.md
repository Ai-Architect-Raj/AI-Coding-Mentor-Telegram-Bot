# AI Coding Mentor Telegram Bot

An AI-powered Telegram bot built using Python, OpenRouter and Telegram Bot API.

Features:
- Answer coding questions
- Teach Python, Java, C++, SQL
- Explain DSA
- Explain Mathematics
- AI/ML guidance
- Interview preparation

Tech stack:
- Python
- Telegram Bot API
- OpenRouter API
- GitHub

Future plans:
- Chat memory
- Voice support
- Multi-model AI
- Deployment
- Resume builder
- GitHub integration
